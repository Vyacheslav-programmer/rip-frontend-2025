# Лабораторная 6: Adaptive PWA Tauri

## Студент: Пронин Вячеслав Константинович, ИУ5Ц-73Б

## Цель работы:
Внедрение адаптивности и развертывание приложения

## Задачи:
- Добавление Redux Toolkit для управления состоянием фильтров
- Реализация адаптивного дизайна для 3 страниц
- Создание PWA (Progressive Web App)
- Разработка нативного приложения на Tauri
- Развертывание на GitHub Pages с HTTPS

## Технологии:
- Redux Toolkit
- PWA (Service Workers, Manifest)
- Tauri (Rust + WebView)
- GitHub Pages
- Адаптивный CSS

## Основные функции:
- Сохранение состояния фильтров в Redux
- Адаптивная верстка для мобильных устройств
- Оффлайн-режим через PWA
- Нативное приложение через Tauri
- Развертывание на GitHub Pages

## Запуск:
# React приложение
npm install
npm run dev

# Tauri приложение
cd src-tauri
cargo tauri dev
